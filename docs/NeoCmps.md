# NeoCRM 组件类型汇总
本文档汇总了 NeoCRM 平台中注册的组件类型，按功能分类整理。

## 表单输入组件: 可作为属性配置项
- **neoInput** - Input 输入框
- **neoInputNumber** - InputNumber 数字输入框
- **neoInputTree** - InputTree 树形输入框
- **neoSearch** - Search 搜索框
- **neo_select** - Select 选择器
- **neoCustomSelect** - CustomSelect 自定义选择器
- **neoTreeSelect** - TreeSelect 树形选择器
- **neoCascader** - Cascader 级联选择器
- **neoAutoComplete** - AutoComplete 自动完成
- **neoMentions** - Mentions 提及
- **neoDatePicker** - DatePicker 日期选择器
- **neoTimePicker** - TimePicker 时间选择器
- **neoSwitch** - Switch 开关
- **neoRadio** - Radio 单选框
- **neoCheckbox** - Checkbox 复选框
- **neoCheckboxGroup** - CheckboxGroup 复选框组
- **neoRate** - Rate 评分
- **neoSlider** - Slider 滑动输入条
- **neoUpload** - Upload 上传
- **neoRichText** - RichText 富文本编辑器
- **neoEditor** - Editor 编辑器

## 数据展示组件
- **neoTable** - Table 表格
- **neoTree** - Tree 树形控件
- **neoTreeGrid** - TreeGrid 树形表格
- **neoList** - List 列表
- **neoDescriptions** - Descriptions 描述列表
- **neoStatistic** - Statistic 统计数值
- **neoCountdown** - Countdown 倒计时
- **neoProgress** - Progress 进度条
- **neoSkeleton** - Skeleton 骨架屏
- **neoEmpty** - Empty 空状态
- **neoResult** - Result 结果页
- **neoAlert** - Alert 警告提示
- **neoBadge** - Badge 徽标数
- **neoTag** - Tag 标签
- **neoAvatar** - Avatar 头像
- **neoAvatarGroup** - AvatarGroup 头像组
- **neoImage** - Image 图片
- **neoIconView** - IconView 图标视图
- **neoTimeline** - Timeline 时间轴
- **neoSteps** - Steps 步骤条
- **neoCarousel** - Carousel 走马灯
- **neoCalendar** - Calendar 日历
- **neoCollapse** - Collapse 折叠面板
- **neoCard** - Card 卡片
- **neoSpace** - Space 间距
- **neoDivider** - Divider 分割线
- **neoBreadcrumb** - Breadcrumb 面包屑
- **neoBackTop** - BackTop 回到顶部
- **neoAnchor** - Anchor 锚点

## 导航组件
- **neoMenu** - Menu 导航菜单
- **menuIcon** - MenuIcon 菜单图标
- **menuButton** - MenuButton 菜单按钮
- **neoTabs** - Tabs 标签页
- **neoPagination** - Pagination 分页
- **neoPageHeader** - PageHeader 页头

## 反馈组件
- **neoModal** - Modal 对话框
- **neoMessage** - Message 全局提示
- **neoNotification** - Notification 通知提醒框
- **neoPopover** - Popover 气泡卡片
- **neoPopconfirm** - Popconfirm 气泡确认框
- **neoTooltip** - Tooltip 文字提示
- **neoSpin** - Spin 加载中
- **neoDropdown** - Dropdown 下拉菜单

## 布局组件
- **neoLayout** - Layout 布局
- **neoContainer** - Container 容器
- **neoGrid** - Grid 栅格
- **neoFeatrueTip** - FeatrueTip 功能提示

## 业务组件
- **NeoPage** - NeoPage 页面组件
- **NeoGrid** - NeoGrid 网格组件
- **NeoGridNew** - NeoGridNew 新版网格组件
- **neoGridEditing** - GridEditing 网格编辑
- **neoPopCheck** - PopCheck 弹出选择
- **CustomNameField** - CustomNameField 自定义名称字段
- **neoBulkEditDialog** - BulkEditDialog 批量编辑对话框
- **neoSelector** - Selector 选择器
- **neoMap** - Map 地图
- **neoIframe** - Iframe 内嵌框架
- **neoTransfer** - Transfer 穿梭框
- **neoLayoutError** - LayoutError 布局错误

## 其他组件
- **neoButton** - Button 按钮
- **neoCustomMessage** - CustomMessage 自定义消息
- **neoActionSheet** - ActionSheet 动作面板
- **neoAffix** - Affix 固钉
